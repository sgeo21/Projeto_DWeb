# 🐾 Amor de Patas

## 📋 Sobre o Projeto

**Amor de Patas** é um site institucional fictício desenvolvido para uma ONG de adoção de animais fictícia. O objetivo da plataforma é divulgar animais disponíveis para adoção, apresentar a missão da organização e facilitar o processo de solicitação de adoção por meio de um formulário simples e intuitivo.

O projeto foi desenvolvido com foco em uma interface amigável, acessível e responsiva, proporcionando uma experiência agradável para usuários interessados em encontrar um novo companheiro de quatro patas.

---

## 🐶 Sobre a ONG

A **Amor de Patas** é uma ONG dedicada ao resgate, cuidado e adoção responsável de cães e gatos em situação de abandono, maus-tratos ou vulnerabilidade.

Nossa missão é oferecer uma nova chance para animais que precisam de amor, segurança e um lar acolhedor. Trabalhamos com voluntários apaixonados pela causa animal e contamos com o apoio da comunidade para transformar vidas.

Além dos resgates, promovemos campanhas de conscientização, vacinação, castração e adoção responsável, contribuindo para um futuro melhor para os pets da nossa região.

---

## ✨ Funcionalidades

- Página institucional apresentando a ONG.
- Catálogo de animais disponíveis para adoção.
- Contador de pets cadastrados.
- Botões de adoção rápida para cada animal.
- Formulário de solicitação de adoção.
- Validação de campos obrigatórios.
- Máscara para telefone.
- Modal de confirmação após envio da solicitação.
- Layout responsivo para diferentes dispositivos.
